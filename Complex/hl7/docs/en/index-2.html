2. Installation

2.1 How to obtain HL7_lib

The latest version will be available at Source Forge.

2.2 Requirements

This library is self contained and attempts to be portable. You should need no other libraries or special installation methods.

2.3 Compilation and installation

In order to compile and install HL7_lib on your system, type the following in the base directory of the HL7_lib distribution: 

% ./configure
% make
% make install

Since HL7_lib uses 

autoconf
you should have not trouble compiling it. Should you run into problems please report them to the the author at James M. Rogers
